# snapdrop

[![Build](https://github.com/gsmlg-ci/snapdrop/actions/workflows/build.yml/badge.svg)](https://github.com/gsmlg-ci/snapdrop/actions/workflows/build.yml)

Docker image published to:
- `docker.io/gsmlg/snapdrop`
- `ghcr.io/gsmlg-dev/snapdrop`

## Usage

```bash
# From Docker Hub
docker pull gsmlg/snapdrop:latest

# From GitHub Container Registry
docker pull ghcr.io/gsmlg-dev/snapdrop:latest
```

## Build

```bash
docker build -t gsmlg/snapdrop .
```
